Bem Vindo


O sistema de controle de consultas irá lhe proporcionar uma visualização mais ampla dos dados em questão.

No link pacientes poderá visualizar, editar e excluir as consultas, para incluir clique no botão incluir que esta localizado no final da tabela.

Para ter um relatório mais complexo das consultas, clique no link relatório. Nessa página clique no botão buscar e coloque os dados que gostaria de filtrar sua consulta, será gerado um relatório do que foi solicitado, caso queira gerar um arquivo do excel de sua busca, clique no link (X em verde) que se encontra no final da tabela.

No rodapé do site você temos links rápidos para sites mais usados, hotmail, facebook, twitter, gmail, e o ultimo para deslogar do sistema.
